Setting up for Lazy:
In the plugins/malpha.lua file, we add the commands:

return {
	"tkachenkosi/malpha.nvim",
	config = function()
			require("malpha").setup({
        title = "Start screen",
				count_recent = 15,
			})
			require("malpha").enable_autostart()
	end,
	event = "VimEnter",
}

q      - close start screen and quit nvim
CR     - open the selected buffer
